# Meme Reference Library

This file maps well-known meme universes into reusable joke mechanics.

Do not copy copyrighted frames, characters, dialogue, logos, or exact scene
compositions. Use these as emotional structures and remix them into original
images, captions, and platform-native layouts.

## Attack on Titan / 进击的巨人 Style

Core value: not just "giant breaks wall". The useful meme space is despair,
reversal, military seriousness, betrayal, hidden truth, and absurd overreaction.

### Reusable Meme Mechanics

#### Wall Break / 墙没了

Use when:
- a small-looking issue suddenly becomes existential
- tech debt, user feedback, or production bugs arrive all at once

Caption seeds:
- 我：这个 bug 应该不大 / bug：
- 产品经理：就改个小需求 / 代码库：
- 上线前：一切正常 / 上线后：

Safe visual:
- original city wall, huge shadow, tiny engineers/users below
- no copied titan design or recognizable characters

#### Basement Reveal / 地下室真相

Use when:
- the team finally opens logs/analytics and discovers a worse truth
- a user research result destroys the original assumption

Caption seeds:
- 我们终于打开数据后台 / 发现根本没人点
- 以为问题是功能不够 / 结果是没人看懂
- 打开用户反馈之前 / 打开用户反馈之后

Safe visual:
- dusty basement door, glowing dashboard inside, shocked tiny figure

#### Rumbling / 地鸣

Use when:
- requirements, notifications, issues, or tasks roll in unstoppably
- a trend overwhelms a small creator/project

Caption seeds:
- 我刚发完第一版 / issue 开始地鸣
- 以为只有一个需求 / 结果需求自己带了全家
- GitHub Trending 一刷新 / 我的小项目开始发抖

Safe visual:
- giant abstract silhouettes marching like unread notifications

#### Reiner Breakdown / 莱纳坐下

Use when:
- someone is visibly exhausted but still pretending to be professional
- open-source maintainers, founders, students, creators, devs

Caption seeds:
- 表面：欢迎提 issue / 内心：别再提了
- 我：这个需求可以做 / 也是我：先让我坐一会
- 维护开源项目前 / 维护开源项目后

Safe visual:
- tired armored office worker sitting under fluorescent light

#### Levi Cleaning / 兵长洁癖

Use when:
- refactoring, linting, cleanup, formatting, deleting bad code
- obsessive polishing before shipping

Caption seeds:
- 用户要功能 / 我先格式化全项目
- 发版前最后一步 / 顺手重构一切
- 说好只修一个 bug / 结果清理了三小时

Safe visual:
- tiny serious cleaner in a messy code room, no recognizable character

#### Survey Corps Charge / 调查兵团冲锋

Use when:
- a team knowingly ships into chaos
- creators publish despite low confidence

Caption seeds:
- 明知道没人看 / 还是发了
- 明知道会被喷 / 还是开源了
- 版本 0.1 发布现场

Safe visual:
- tiny team charging toward a huge dashboard/error wall

#### "Freedom" Obsession / 自由

Use when:
- someone claims freedom but is trapped by tools, metrics, or algorithms

Caption seeds:
- 我追求的是自由 / 不是每天看数据
- 说好自由创作 / 结果被算法拿捏
- 做工具是为了自由 / 现在天天求 star

Safe visual:
- person reaching toward sky while tangled in notification wires

#### "Ten Years At Least" / 十年起步

Use when:
- exaggerated attachment, parasocial project obsession, sunk cost

Caption seeds:
- 这个项目没人用也要维护 / 至少十年吧
- 希望用户记住我 / 至少十年吧
- 我对这个破 skill 的执念 / 至少十年吧

Safe visual:
- melodramatic creator kneeling beside a laptop, no copied panel

#### Potato Girl / 土豆名场面

Use when:
- someone is eating/snacking at the worst possible formal moment
- creator unseriousness in serious contexts

Caption seeds:
- 项目复盘会 / 我在想晚饭
- 大家讨论商业模式 / 我在做表情包
- 开源发布前夜 / 先吃一口

Safe visual:
- nervous student/dev holding a snack in a formal meeting room

### Good Prompt Pattern

```text
Create an original meme image with Attack-on-Titan-like emotional mechanics:
overwhelming scale, tiny humans, military seriousness, sudden disaster, but no
recognizable characters, costumes, logos, titans, or copied scenes.
Scene: <your product/user/bug situation>.
Leave blank caption space at top and bottom.
```

### Anti-Patterns

- Explaining the lore instead of making the joke
- Only using "giant breaks wall"
- Copying anime frames or recognizable characters
- Making it too polished and heroic
- Adding too much text; AOT-style memes work best with one short setup and one image punchline

## Chinese Tech Meme Angles

Use these when combining AOT mechanics with AI/open-source/product scenes:

- 用户反馈 = 巨人
- README = 墙
- issue = 地鸣
- 数据后台 = 地下室
- 维护者 = 莱纳坐下
- 格式化/重构 = 兵长洁癖
- 发版 = 调查兵团冲锋
- 算法/流量 = 自由的反面
